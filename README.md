# AI Language Translation Tool 🌐

A clean, responsive, and instant Language Translation Tool built as part of the **CodeAlpha Software Development Internship** (Task 1).

## 🚀 Live Demo
Aap yahan apni live GitHub Pages ki link paste kar dena (e.g., https://yourusername.github.io/AI-Language-Translation-Tool/)

## ✨ Features
- **Instant Translation:** Seamless translation between multiple languages using the MyMemory API.
- **Modern UI:** Built with HTML5 and Tailwind CSS for a clean and premium look.
- **Text-to-Speech:** Listen to both the source and the translated text with a single click.
- **Quick Actions:** Swap languages instantly and copy text to the clipboard with dedicated utility buttons.

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Web Speech API (for Text-to-Speech integration)
